7. Performance Decisions

Your priorities were simplicity and speed.

So this design intentionally avoids:

Thing	Why
React / frameworks	unnecessary
external UI libraries	slower
map rendering	unnecessary
sorting client-side	server already sorts
heavy icons/fonts	slower mobile load

Total JS payload is only a few KB.

8. Mobile Optimization
